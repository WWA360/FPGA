基于FPGA的HC-SR04超声波测距系统
一、项目简介
本项目基于 FPGA 平台开发，采用 HC-SR04 超声波测距模块，实现实时障碍物距离检测、数码管动态距离显示、蜂鸣器梯度报警功能。系统测距响应迅速、稳定性高，可实现近距离防撞预警效果，适合嵌入式FPGA基础实训、智能测距应用场景。
二、核心功能
- 超声波测距采集：FPGA 驱动 HC-SR04 模块，通过发射超声波信号、接收回波信号，精准计算障碍物与设备的实际距离，有效测距范围适配常规实训场景。
- 数码管实时显示：将检测到的距离数值实时刷新在数码管上，直观展示当前测距数据，单位为厘米(cm)。
- 蜂鸣器梯度报警（核心特色）：系统根据测距距离动态调节蜂鸣器鸣叫频率：障碍物距离越近，蜂鸣器鸣叫越急促、频次越高；距离越远，蜂鸣器声音越平缓，实现距离分级预警，人机交互体验更强。
三、硬件组成
- FPGA 主控芯片（项目核心控制单元）
- HC-SR04 超声波测距模块（距离采集）
- 多位数码管（数据可视化显示）
- 有源/无源蜂鸣器（梯度声光预警）
四、工程结构
本工程代码模块化拆分，结构清晰，便于二次修改与学习：
- bell：蜂鸣器驱动模块，实现变频鸣叫逻辑
- digi_tube：数码管驱动模块，负责距离数值解码与显示
- ultrasonic：超声波测距核心模块，完成信号触发、回波检测、距离换算
- prj：FPGA 工程配置、编译缓存与输出文件
- doc：项目相关文档资料
五、工作原理
1. FPGA 定时发送触发信号，驱动 HC-SR04 发射超声波信号；
2. 超声波接触障碍物后反射，模块接收回波并返回回响信号；
3. FPGA 检测回响信号高电平持续时间，结合声速公式换算出精准距离；
4. 将距离数据送入数码管模块，实时刷新显示；
5. 系统根据实时距离阈值动态调整蜂鸣器频率，实现近急远缓的智能报警效果。
六、项目特点
- 代码模块化设计，可读性高、移植性强
- 测距实时响应，显示稳定不闪烁
- 梯度蜂鸣报警，区别于固定频率报警，功能更智能
- 纯Verilog硬件逻辑实现，无软件延时，运行高效稳定
